Uses JSON library, https://github.com/stleary/JSON-java and JSON simple library, https://github.com/fangyidong/json-simple. To run the program, add both libraries to your classpath, making sure you download and save the config.json file in the same file as ReadInArrayFromJSON.java.
If .json files being parsed are not encoded in ANSI or UTF-8, the program will crash due to unexpected formatting.
